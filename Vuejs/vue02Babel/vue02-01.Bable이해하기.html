Babel 에 대해서 학습한다.

        Babel 이란 JS 컴파일러(=번역기)다.
            ES2016 를 ES5 로 바꾸어 주는
            JSX 를 ES5로 바꾸어 주는

        온라인 babel 컴파일러 : https://babeljs.io/repl/

        


        변수선언:
            ES5 : var        : 함수 스코팅
            ES2016 : let, const : 블락 스코핑
        함수선언:
            ES5 : function(){}
            ES2016 : ()=>{ }
