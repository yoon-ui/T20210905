npm 사용법
React CLI 을 사용하여 Crud 어플리케이션을 만들어 본다. 1. 프로젝트 생성 작업 step 1-01. 프로젝트 생성 $ npx create-react-app crudreact # 프로젝트명은 소문자로 만들어야 함. $ ls # crudreact 폴더가 보이는지 확인 $ cd crudreact # crudreact 폴더로 이동 $ ls # package.json 파일이 보이는지 확인 $ npm run start # package.json 파일이 보이면 서버 실행 step 1-02. 프로젝트 root에 .env 파일을 만들고 아래 코드를 추가 한다. SKIP_PREFLIGHT_CHECK=true step 1-03. install the Bootstrap 4 $ npm install --save bootstrap step 1-04. 라우팅 처리 $ npm install --save react-router-dom history react-router-redux step 1-05. install React redux $ npm install --save redux react-redux redux-logger $ npm install --save-dev redux-devtools step 1-06. install material-ui $ npm install --save @material-ui/core step 1-07. 비동기 처리 $ npm install --save redux-saga redux-thunk axios step 1-08. immer 설치 $ npm install immer --save step 1-09. reselect 설치 $ npm install --save reselect step 1-11. 프로젝트 실행 $ ls # package.json 보이는지 확인 $ npm run start # package.json 보이면 서버 실행 step 1-12. 코드 표준화 검사 $ npm run lint step 1-13. 배포용 파일 빌드. /dist/ 폴더에 파일 만들어짐. $ npm run build 2. 프로젝트 폴더 구조 프로젝트 ├─ dist # 빌드 결과물 폴더 ├─ node_modules # npm 모듈(패키지) 저장 폴더 ├─ public # 배포 버전을 빌드할 때 사용하는 파일. │ └─ index.html │ └─ favion.ico └─ src # 소스 코드 폴더 │ └─ assets # 기타 자원 │ │ └─ logo.png │ ├─ components # 컴포넌트 │ ├─ store # │ ├─ router # 라우터 │ ├─ reducers # │ ├─ utils # │ ├─ views # 라우터 페이지 │ ├─ App.js │ └─ index.js ├─ package.json └─ README.md 3. 프로젝트 폴더 및 파일 생성 3.1 폴더 생성 $ mkdir -p ./src/assets \ ./src/assets/images \ ./src/assets/style \ ./src/assets/text \ ./src/common/mergeReducers.js \ ./src/reducers/index.js \ ./src/store/index.js 3.3 프로젝트 컴포넌트 파일 생성 $ mkdir -p ./src/components/timelines \ ./src/components/frineds \ ./src/reducers \ ./src/views $ touch ./src/components/timelines/TimelineList.js \ ./src/components/frineds/FriendList.js \ ./src/reducers/timeline.js \ ./src/reducers/frined.js \ ./src/views/TimelineMain.js \ ./src/views/FriendMain.js \ ./src/store/index.js 4. 프로젝트 초기 설정. 프로젝트의 폴더 구조 └─ public/index.html 에 설정 4-1. fontawesome 설정 <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.0.12/css/all.css"> 4-2. ubuntu 폰트 설정 <link href="https://fonts.googleapis.com/css?family=Ubuntu" rel="stylesheet"> 4-3. bootstrap 설정 <link
