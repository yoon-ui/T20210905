hook이 뭔가요?

hook은 함수 컴포넌트에서 state와 생명주기(lifecycle)을 "연동(hook into)"할 수 있게 해주는 함수입니다.
hook은 함수형 컴포넌트 안에서만 동작합니다.

읽기 전용 변수

useEffect (★)

부수 효과 처리.

부수 효과란 함수 외부의 데이터를 변경하는 연산.

부수 효과 함수는 real dom 에 반영된 후 비동기로 호출

==> mount 나 update 후에

==> unmount 전에

부수 효과 함수에는 이벤트 핸들러 등록 및 해제, 외부 API 호출등

의존성 배열 있으나 사용하지 말자

사용자 훅:

이름이 use 로 시작하로록 명명

훅 사용시 규칙

1. 하나의 컴포넌트에서 혹을 호출하는 순서는 항상 같아야 한다.
2. 훅은 함수형 컴포넌트 또는 커스텀 훅 안에서 호출되어야 한다.

함수형 컴포넌트에서 불필요한 렌더링을 막아라

  1. useEffect 사용해서
  2. useReducer 사용해서
  3. useRef 사용해서

의존성 배열 있으나 사용하지 말자. 연관 배열 대체 방법

  1. useEffect() 나 useCallback() 안에서 if 문 사용
  2. 상태값 변경 메서드의 인자로 함수 사용
  3. useRef()
